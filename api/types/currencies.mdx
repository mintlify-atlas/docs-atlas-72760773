## ECurrency

The `ECurrency` enum defines all supported currencies on the GRVT platform. These include fiat currencies, stablecoins, major cryptocurrencies, and various altcoins.

### Fiat Currencies

| Currency | Value | Description |
|----------|-------|-------------|
| USD | `'USD'` | US Dollar fiat currency |

## Usage Example

```typescript
import { ECurrency } from '@grvt/sdk';

// Use currency enum in your code
const currency = ECurrency.USDC;
console.log(currency); // 'USDC'
```

<Note>
The "K" prefix on some tokens (KPEPE, KSHIB, KBONK) represents 1000x scaling, commonly used for tokens with very small individual values.
</Note>

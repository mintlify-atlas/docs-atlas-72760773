## Overview

The `getFundingAccountSummary()` method retrieves a summary of your funding account, including balance information and account details.

## Method Signature

```typescript
await client.getFundingAccountSummary(): Promise<IApiFundingAccountSummaryResponse>
```

## Parameters

This method does not accept any parameters.

## Response

<ResponseField name="IApiFundingAccountSummaryResponse" type="object">
  The funding account summary response
  
  <Expandable title="Response fields">
    <ResponseField name="account_id" type="string">
      The funding account identifier
    </ResponseField>
    
    <ResponseField name="balances" type="array">
      Array of balance objects for different currencies
      
      <Expandable title="Balance object">
        <ResponseField name="currency" type="string">
          The currency code (e.g., "USDT", "BTC")
        </ResponseField>
        
        <ResponseField name="available" type="string">
          Available balance for the currency
        </ResponseField>
        
        <ResponseField name="total" type="string">
          Total balance including locked funds
        </ResponseField>
        
        <ResponseField name="locked" type="string">
          Funds currently locked in pending operations
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="total_value_usd" type="string">
      Total account value in USD
    </ResponseField>
  </Expandable>
</ResponseField>

## Example

```typescript
import { GrvtClient, EGrvtEnvironment } from '@grvt/client';

const client = new GrvtClient({
  apiKey: 'your-api-key',
  apiSecret: 'your-api-secret',
  tradingAccountId: 'your-account-id',
  env: EGrvtEnvironment.TESTNET
});

try {
  const summary = await client.getFundingAccountSummary();
  
  console.log('Account ID:', summary.account_id);
  console.log('Total Value (USD):', summary.total_value_usd);
  
  summary.balances.forEach(balance => {
    console.log(`${balance.currency}:`);
    console.log(`  Available: ${balance.available}`);
    console.log(`  Total: ${balance.total}`);
    console.log(`  Locked: ${balance.locked}`);
  });
} catch (error) {
  console.error('Error fetching funding account summary:', error);
}
```

## Error Handling

```typescript
try {
  const summary = await client.getFundingAccountSummary();
  // Process summary
} catch (error) {
  if (error.response) {
    // API returned an error response
    console.error('API Error:', error.response.status, error.response.data);
  } else if (error.request) {
    // Request was made but no response received
    console.error('Network Error:', error.message);
  } else {
    // Other error
    console.error('Error:', error.message);
  }
}
```
